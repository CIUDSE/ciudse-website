## Development Guide

### 0. Prerequisites
- Git
- Yarn

### 1. Clone repository
```
git clone https://github.com/CIUDSE/ciudse-website.git
cd ciudse-website
```

### 2. Install dependencies

```
yarn install
```

### 3. Run a local development server

```
yarn start
```

If you want to build a production version of the website

```
yarn build
```
